# Mood-Palette-Generator
A small web app that creates color palettes based on the user's mood.

## Core Concepts:
- HTML, CSS, JavaScript fundamentals
- DOM manipulation
- Working with array and randomization
- UI/UX design basics

## Tools:
- VS Code
- Browser DevTools
- Maybe TailwindCSS

## Features:
- User picks a mood (calm, happy, sad, angry, etc...)
- App generates a 5-colour palette (random or rule-based)
- Each colour shows its HEX code and a "copy" button
